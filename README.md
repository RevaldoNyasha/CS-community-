DEV-CRAFT is a professional Computer Science community platform built to help students collaborate, access academic resources, and stay updated with hackathons and technical events.

The platform is designed to replace scattered class groups, lost resources, and informal communication channels with a structured, moderated, and academic-focused system.

It allows students to:

Access past study materials

Request missing resources

Participate in community discussions

View upcoming and past hackathons

See hackathon winners and projects

Interact in a clean, professional environment

The system also includes an Admin panel for content moderation and platform management.
